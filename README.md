# local-typeless

Native macOS menu-bar dictation app. Offline speech-to-text (EN + ZH) with local LLM polish, inspired by Typeless.

## Build

    make build
    make test
    make run

## Requirements

- macOS 14+
- Xcode 15.3+ (Swift 5.9)
- [XcodeGen](https://github.com/yonaskolb/XcodeGen): `brew install xcodegen`
